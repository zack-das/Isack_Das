A high-performance, responsive personal portfolio built with React and Custom CSS. This project showcases a "Cyber-Dark" aesthetic with a focus on fluid typography and minimalist design.

🎨 Theme & Aesthetic
The website utilizes a custom color palette designed for high visibility and a tech-centric feel:

Primary Background: #050a09 (Deep Charcoal/Black)

Accent Cyan: #00fff2 (Used for highlights and branding)

Deep Blue: rgb(0, 122, 192) (Used for subheadings and UI elements)

Neon Green: rgb(38, 192, 0) (Used for interactive social icons)

🚀 Key Technical Features
Fluid Typography: Implementation of clamp() functions ensures that headers and text scale perfectly from mobile devices to ultra-wide monitors without breakpoints.

Custom CSS Architecture: 100% custom styling—no external CSS frameworks (Tailwind/Bootstrap) to ensure a lightweight footprint and unique identity.

Interactive UI Elements: * Hover-animated social icons with directional transitions.

Glassmorphism effects via backdrop-filter on profile elements.

Custom-built vertical scroll indicators.

🛠️ Built With
React 18 - Frontend Library

Custom CSS3 - Featuring Flexbox and Advanced Selectors

React Icons - For social and navigation iconography

Google Fonts - Integrated system-wide
